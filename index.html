What is habit stacking?
Clear's formula: "After I [current habit], I will [new habit]." By chaining habits together, each one becomes the cue for the next — your routine runs on autopilot instead of willpower. The stack order below determines what appears on your Dashboard. Drag habits into the sequence that matches your actual day.

Google Drive Setup (5 minutes, free)
1. Go to console.cloud.google.com
2. Create a new project (name it anything)
3. Go to APIs & Services → Library
4. Search for Google Drive API → Enable it
5. Go to APIs & Services → Credentials
6. Click Create Credentials → OAuth Client ID
7. Application type: Web application
8. Under Authorized JavaScript origins, add the URL of this file (e.g. file:// or your hosted URL)
9. Copy the Client ID and paste it above
10. Click Connect Drive

Habit Scorecard
Why score a habit as bad here? The scorecard in Atomic Habits is a neutral audit tool — you first list ALL current habits and label them +, =, or – before deciding what to change. Scoring a habit as "–" here flags it in your database so you can see the full picture of your current system. Use the Bad Habits tab to apply the 4 Inverse Laws and build a dismantling strategy for it.
